# Altseed2-csharp

## Altseed2

 [Altseed](https://github.com/altseed/Altseed) の後継となるべく開発しているゲームエンジンです。

各種機能を内部実装する [Core](https://github.com/altseed/Altseed2) と それをラップし C# で使いやすくする Engine (本リポジトリ) に分かれています。
